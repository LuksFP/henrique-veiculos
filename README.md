# Henrique Veiculos

Reconstrucao estatica do site da Henrique Veiculos.

## Rodar localmente

```bash
cd /home/lukas/henrique-veiculos
python3 -m http.server 8082
```

Depois acesse `http://127.0.0.1:8082/`.

## Observacao

O dominio publico bloqueou captura direta via CloudFront. A pagina foi reconstruida com base no conteudo indexado publicamente e usa placeholders visuais proprios no lugar das fotos/logos originais.
